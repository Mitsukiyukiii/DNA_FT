This program takes a dna sequence from file (fasta format), generates and tehn plots the Fourier spectrum of that sequence by the method outlined in the following publication:

Tiwari, S., Ramachandran, S., Bhattacharya, A., Bhattacharya, S. & Ramaswamy, R. Prediction of probable genes by Fourier analysis of genomic sequences. Comput. Appl. Biosci. 13, 263–270 (1997).
